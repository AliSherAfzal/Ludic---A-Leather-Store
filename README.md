# About

StrideX is a premium leather footwear e-commerce platform designed to showcase modern software engineering and DevOps practices.

Rather than being built as a tutorial project, every feature follows a professional engineering workflow:

**Business Requirement → UX Design → Architecture → Implementation → Code Review → Documentation → Git**

The project serves as a long-term portfolio demonstrating clean architecture, scalable frontend development, backend engineering, CI/CD, containerization, cloud deployment, and infrastructure automation.

# Quick Start

### Clone

```bash
git clone <repository-url>
cd ludic
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Application:

```text
http://localhost:5173
```

### Backend

```bash
cd backend
npm install
npm run dev
```

Health Endpoint:

```text
GET /health
```
